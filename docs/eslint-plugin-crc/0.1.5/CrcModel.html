CrcModel

CrcModel

Represents a Class-Responsibility-Collaboration model, which expresses the scope of an object's behaviors (responsibilities) and the objects it depends on to fulfill its responsibilities (collaborators).

Constructor

new CrcModel(name, optionsopt)

Create a shadow Class-Responsibility-Collaboration model.

Parameters:
Name Type Attributes Description
name string

The source code identifier of the class or object to be modeled.
options object <optional>

A parameter object that optionally sets all of the CrcModel's properties.

Properties
Name Type Attributes Description
collaborators array <optional>

A collection of zero or more CrcModels that this object depends on.
references array <optional>

A collection of zero or more Espree Tokens representing where this object is used in the source code.
identifier object <optional>

An Espree Identifier object.
responsibilities array <optional>

A list of data this object must maintain and/or operations it must perform.
declaration object <optional>

The Espree Identifier that represents "where" this object was declared into being.
superClass object <optional>

The prototype of the object being modeled. Defaults to Object.
